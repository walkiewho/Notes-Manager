# Notes-Manager

Простое веб-приложение для заметок с автоматическим сохранением в браузере.

## Возможности

- Создание заметок с текстом
- Автоматическое сохранение при редактировании
- Удаление отдельных заметок
- Очистка всех заметок одной кнопкой
- Все данные сохраняются в localStorage и не теряются после перезагрузки страницы
